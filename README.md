# last-management

** หมายเหตุ ** ติดดั้ง yarn,nodejs,npm,git เพื่อใช้ในการ run project

## install package
```javascript
yarn
```
## run project
```javascript
//mode development

yarn dev

//mode production

yarn start
```

## deploy
```javascript
// เลือกไฟล์ทั้งหมดที่แก้ไข
git add . 

// อธิบาย หรือรายละเอียดการอัพเดต
git commit -m "ข้อความ"

// นำโค๊ดขึ้น server
git push
```
